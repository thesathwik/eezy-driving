# EAZY Driving School Platform

A full-stack web application for connecting learner drivers with driving instructors.

## Tech Stack

- **Frontend**: React, React Router, Stripe Elements
- **Backend**: Node.js, Express
- **Database**: MongoDB Atlas
- **Payment**: Stripe
- **Maps**: Google Maps API

## Getting Started

### Installation

1. Install dependencies:
```bash
npm install
cd server && npm install
```

2. Configure environment variables (copy from .env.example)

3. Run development:
```bash
# Frontend (port 3000)
npm run dev

# Backend (port 5001)  
npm run server:dev
```
